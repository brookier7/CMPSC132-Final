# CMPSC132-Final
This project is a number guessing game. The computer generates a number 1-100 and the player has to guess the number, taking into account feedback from the computer.

To use the program, first guess any valid number. The computer will give you information on how accurate your guess is. Using this information, keep guessing until you get the number at which point the computer will tell you that you've won and how many guesses it took.
